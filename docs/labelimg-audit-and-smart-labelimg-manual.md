This document records the LabelImg reference audit, the practical operating manual, and the current Smart LabelImg comparison.

- Official repository: https://github.com/HumanSignal/labelImg

## LabelImg Feature List

LabelImg is a Qt image annotation tool for rectangular object boxes. Its core workflow is opening an image or image folder, drawing boxes, assigning labels, and saving annotation files.

Supported annotation formats:

- Pascal VOC XML.
- YOLO TXT.
- CreateML JSON.
- Predefined class list through `data/predefined_classes.txt`.

File and navigation features:

- Open one image.
- Open an image directory.
- Open a separate annotation file.
- Open recent files.
- Change the default save directory.
- Save.
- Save As.
- Switch save format.
- Close current image.
- Delete current image.
- Reset all settings.
- File list panel for quickly jumping between images.
- Previous image and next image navigation.
- Copy bounding boxes from the previous image.

Box editing features:

- Create rectangle boxes.
- Edit/select boxes.
- Move boxes with mouse.
- Move selected box with arrow keys.
- Resize boxes.
- Delete selected box.
- Duplicate selected box.
- Context-menu copy or move shape after dragging with right mouse.
- Optional square/rectangle drawing constraint.
- Beginner and advanced editing modes.

Label/class features:

- Choose labels from a class list.
- Edit selected box label.
- Delete label/box from the label list.
- Single-class mode.
- Default label/class mode.
- Display or hide label text on boxes.
- Filter/show objects by label through the label combo box.
- Mark objects as difficult.
- Label colors generated by label text.
- Change global box line color.
- Change selected box line/fill color.

View features:

- Show all boxes.
- Hide all boxes.
- Show/hide label dock.
- Zoom in.
- Zoom out.
- Original size.
- Fit window.
- Fit width.
- Brightness increase.
- Brightness decrease.
- Brightness reset.

Dataset review features:

- Mark image as verified.
- Verified images show a visual state.
- Auto save mode when navigating.

Help and utility features:

- Shortcut help dialog.
- Tutorial dialog.
- About/info dialog.
- Docker/source/PyPI install paths.
- PyInstaller packaging instructions.

## LabelImg Operation Manual

### Pascal VOC XML Workflow

1. Start LabelImg.
2. Choose the annotation output folder with `Change default saved annotation folder`.
3. Choose `Open Dir` to load an image folder, or `Open` to load one image.
4. Make sure the save format is Pascal VOC.
5. Press `W` or click `Create RectBox`.
6. Left-drag around the target object.
7. Enter or choose the class name.
8. Select a box to move or resize it.
9. Use arrow keys for small movement.
10. Press `Ctrl+S` to save, or navigate to the next image when auto save is active.
11. The output is one `.xml` file per image.

### YOLO TXT Workflow

1. Prepare a fixed class list before annotation.
2. Start LabelImg.
3. Switch format from Pascal VOC to YOLO.
4. Open one image or a folder.
5. Draw boxes and assign labels.
6. Save each image.
7. LabelImg writes one same-name `.txt` file per image and a `classes.txt` file.
8. Do not reorder classes in the middle of a YOLO dataset, because old YOLO class ids will not be automatically rewritten.

### Opening Existing Labels

1. Put existing labels next to images with matching filenames, or choose `Open Annotation`.
2. For a folder workflow, open the image folder.
3. Select images in the file list.
4. Boxes appear if the matching label file exists.

### Editing Boxes

1. Click a box to select it.
2. Drag inside the box to move it.
3. Drag a handle/corner to resize it.
4. Press arrow keys for small movement.
5. Press `Delete` to delete the selected box.
6. Press `Ctrl+D` to duplicate it.
7. Use the label list to edit or delete the selected object.

### Review And Navigation

1. Use `A` and `D` to move between images.
2. Use `Space` to mark an image as verified.
3. Use `Ctrl+V` to copy boxes from the previous image when consecutive frames/images share the same objects.
4. Use hide/show boxes and display-label options for visual checking.

## Smart LabelImg Current Manual

### Download And Start

1. Download the release zip for your system.
2. Unzip it.
3. macOS: open `Smart LabelImg.app`.
4. Windows: open `Smart LabelImg\Smart LabelImg.exe`.
5. Linux: open `Smart LabelImg`.
6. The release package includes MobileSAM and `models/mobile_sam.pt`; no separate model download is required.
7. For Windows source install and exe builds, see `docs/windows-install-build.md`.

### Basic Opening Logic

1. Click `Open`.
2. Choose `Image` for one image, or `Folder` for an image folder.
3. Click `Open Label`.
4. Choose `Label File` for one `.xml` / `.txt`, or `Label Folder` for a folder of matching labels.
5. When using a label folder, Smart LabelImg matches labels by image basename, for example `images/a.jpg` -> `labels/a.txt` or `labels/a.xml`.

### Save Logic

1. Choose `Save Format`: `YOLO TXT` or `VOC XML`.
2. Click `Save/Target`.
3. Choose `Label File` to save the current image into one explicit file.
4. Choose `Label Folder` to save each image into that folder with the same basename as the image.
5. After a save target is selected, annotation changes auto-save.
6. The old separate `Save` and `Save Target` buttons are merged into one `Save/Target` entry.

### Normal LabelImg Mode

1. Click `普通 LabelImg`, or press `Shift` until the status says normal mode.
2. Drag with the left mouse button to draw a manual box.
3. Click a box to select it.
4. Drag inside a selected box to move it.
5. Drag handles around a selected box to resize it.
6. Press arrow keys to move the selected box by 1 pixel.
7. Right-click or double-click a box to open class/delete actions.

### Smart MobileSAM Mode

1. Click `智能标注`, or press `Shift` until the status says smart mode.
2. Left-drag a rough box around the object.
3. MobileSAM refines the rough box into a tighter object box.
4. Right-click/secondary-click on an object to create a box from a point prompt.
5. Set `Fast Crop Size` to balance speed and accuracy.
6. Use `Full Image` only when object edges are being missed.

### Class Editing

1. Click `Add Class` to add a class.
2. Double-click an existing class to rename it.
3. Renaming a class also updates every existing box that uses the old class.
4. Click `Delete Class` to remove a class.
5. Boxes using a deleted class fall back to `object` if available.

## Smart LabelImg Improvement Status

Implemented after the audit:

- Top opening workflow reduced to `Open` and `Open Label`.
- Removed the previous `Similar` action.
- Added explicit image list panel.
- Added double-click class rename.
- Class rename updates every already-labeled box with that class.
- Added explicit save target file/folder selection.
- Added auto-save after annotation changes.
- Added duplicate selected box.
- Added copy previous image boxes.
- Added label text display toggle.
- Added zoom in, zoom out, and fit window.
- Added verified image tracking.
- Kept MobileSAM smart mode and fast crop settings.

Lower-priority LabelImg parity still open:

- Persist verified state to Pascal VOC XML.
- Object-level difficult flag.
- CreateML JSON export/import.
- Brightness controls.
- Box and label color customization.
- Single-class mode.
- Draw-square constraint.
- Hide/show all boxes button.
- Recent files list.
- Reset all settings.
